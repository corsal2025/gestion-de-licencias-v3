16. Conclusión

Proyecto liviano y eficiente: El código propio del sistema pesa apenas ~350 KB (Core + Infrastructure + Web components). El peso total de 8.61 MB se debe principalmente a Bootstrap (8.28 MB), que es una dependencia estática del frontend.

Dependencias mínimas: Solo 4 paquetes NuGet en runtime. Sin microservicios, sin Docker, sin Redis, sin colas de mensajes. Todo corre en un solo proceso Blazor Server con SQLite local.

Escala: La base de datos SQLite actual pesa 36 KB con datos de prueba. El sistema está diseñado para uso municipal local (una oficina).
